# -0（ゼロ）— 就職者のための企業評価プラットフォーム

公開データを使い、企業の**将来性・生産性・働きやすさ・労働環境リスク**を可視化するWebアプリです。

企業名や単一スコアだけで判断せず、各評価の根拠、利用できたデータ、欠損状況を確認できる設計を目指しています。

## データセット

本アプリには、性質の異なる2種類のデータセットがあります。

### 実データ

実在企業を掲載します。

- 企業名、従業員数、設立年、売上、業種など: Wikidata
- 残業、有給取得率、女性管理職比率など: 厚生労働省「しょくばらぼ」等の公開データ
- 掲載企業数: 約8,300社
- 全国集計: 約14.8万社分の公開データ

実名企業には、取得できていない労働指標や財務指標を推測で付与しません。

### デモデータ

架空企業のサンプルです。

離職率、残業代支給率、ハラスメント、法令違反など、実企業では未連携の指標を含み、ブラック度評価の全機能を確認できます。

**実データ版の企業は実在企業、デモ版の企業だけが架空企業です。**

## 主な評価軸

### 将来性

次の要因から0〜100点で評価します。

- 業種の構造的な見通し
- 売上高の年平均成長率
- 従業員数の年平均成長率
- 企業年齢
- 従業員規模
- 上場の有無

利用可能な要因で生スコアを計算した後、データ欠損が多い企業は50点へ縮約します。

これにより、業種や上場区分など少数の良好指標だけで高評価になることを防ぎます。詳細画面にはデータ充足率と信頼度を表示します。

### 生産性

取得できた財務データから、主に一人当たり売上高を計算します。

- 一人当たり売上高
- 一人当たり営業利益
- 営業利益率

売上高が取得できない企業は、生産性スコアを表示しません。

### 働きやすさ

取得できた次の公開データを使います。

- 月平均残業時間
- 有給取得率
- 3年以内離職率
- 平均勤続年数
- 女性管理職比率

欠損が多い場合は、将来性と同じく中立値へ補正します。

### ブラック度・安全度

離職率、残業代支給率、法令違反など、必要な労働指標が揃った企業だけを評価します。

実名企業について、根拠のないブラック認定は行いません。

## 隠れ優良企業

掲載企業のデータ量に差があるため、次の2種類に分けて抽出します。

### 総合型

- 品質4軸のうち3軸以上を評価可能
- 働きやすさまたは安全度を含む
- 評価可能な軸の平均が基準以上
- 従業員規模が掲載企業の中央値以下

### 成長×労働環境型

- 将来性と、働きやすさまたは安全度を評価可能
- 両方が74点以上
- 従業員規模が掲載企業の中央値以下

財務データがない企業を一律に除外せず、評価できるデータの種類を明示します。

## 相性診断

利用者が重視する軸を選ぶと、企業ごとのマッチ度を計算します。

- 成長したい
- 働きやすさ重視
- ブラック企業を避けたい
- 稼ぐ力を重視
- 規模の安定を重視

選択軸の半数未満しかデータがない企業は比較不能とし、一部欠損の場合は50点へ縮約します。

## データ分析

全国集計と掲載企業群を次の切り口で確認できます。

- 全国の平均残業時間、有給取得率、女性管理職比率、平均年齢
- 業種別ランキング
- 都道府県別ランキング
- 残業時間の分布
- 将来性と生産性の散布図
- 各評価軸の上位企業

## 収益化の状態

Google AdSenseのクライアントIDは所有権確認用に設定されています。

広告スロットIDが未設定の間は、AdSense広告枠を表示しません。サイト上でも「審査コード設置中・広告枠未稼働」と区別して表示します。

アフィリエイトリンクは、プレースホルダーURLを実リンクへ置き換えた案件だけ表示されます。

## 注意事項

- スコアは公開データと編集可能な前提に基づく参考値です。
- 正確性、完全性、最新性を保証するものではありません。
- 就職、転職、投資の助言ではありません。
- 重要な判断では、企業公式情報、有価証券報告書、求人票、公的機関の原典を確認してください。
- データ誤りや企業側からの訂正依頼はGitHub Issuesで受け付けます。
