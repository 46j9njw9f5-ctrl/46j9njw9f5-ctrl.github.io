# 企業評価プラットフォーム「-0（ゼロ）」計画書

就職者のための**適正な企業評価**と**ブラック企業の可視化**を行う Web アプリケーション。

## 1. コンセプト

求人広告やイメージだけでは分からない「実態」を、複数の労働指標から**透明なスコア**で可視化する。
特に**ブラック企業リスク**を定量化し、利用者ができるだけ危険な企業に入らないよう警告する。

- 就職者側: 企業を客観指標で比較し、ミスマッチ・ブラック企業を回避
- 評価の公正さ: スコアの根拠（各指標の寄与）を必ず開示（ブラックボックスにしない）

## 2. スコープ（MVP）

| 機能 | 内容 |
| --- | --- |
| 企業検索 | 名称・業界・地域でのフィルタ／キーワード検索 |
| ブラック度スコア | 労働指標から 0–100 で算出し、4段階のリスク区分で表示 |
| 適正評価 | 良い点・懸念点・総合所見を透明に提示 |
| 企業詳細 | 各指標の実数値、レーダー、赤信号（レッドフラグ）一覧 |
| 企業比較 | 複数社を並べて比較 |
| お気に入り | localStorage に保存（サーバー不要） |
| ソート | ブラック度／ホワイト度／残業／離職率などで並び替え |

## 3. 技術構成

- フロント: Vite + React + TypeScript
- スタイル: 自前CSSデザインシステム（依存を最小化しビルドを安定）
- テスト: Vitest（スコアリングエンジンのユニットテスト）
- データ: リアルな架空企業のシードデータ（`src/data/companies.ts`）
  - データ層を分離し、将来は API / DB / ユーザー投稿へ差し替え可能

## 4. ブラック企業スコアリング（中核ロジック）

各指標を「リスクポイント 0–100」に正規化し、重み付き平均で**ブラック度スコア**を算出。
ホワイト度 = 100 − ブラック度。

| 指標 | 重み | 根拠 |
| --- | --- | --- |
| 月平均残業時間 | 0.20 | 45h超で警告、80h超で過労死ライン |
| 3年以内離職率 | 0.18 | 高いほど定着せず＝実態が悪い |
| 有給休暇消化率 | 0.12 | 低いほど休めない |
| 残業代支給率 | 0.12 | 低い＝サービス残業の疑い |
| 平均勤続年数 | 0.10 | 短い＝使い捨て傾向 |
| ハラスメント指標 | 0.10 | 報告件数（従業員比） |
| 労基署是正勧告 | 0.10 | 直近5年の法令違反歴 |
| 万年採用 | 0.04 | 常時大量採用は高離職の兆候 |
| 社保完備 | 0.04 | 未整備は基礎的な問題 |

### リスク区分

- 0–24: 優良（ホワイト）
- 25–44: 標準
- 45–64: 要注意
- 65–100: ブラック危険

### レッドフラグ（即警告する明示条件）

- 月残業 80h 超（過労死ライン）
- 残業代支給率が低くサービス残業の疑い
- 3年離職率 30% 超
- 労基署の是正勧告あり
- 社会保険 未整備
- 万年採用 かつ 高離職率
